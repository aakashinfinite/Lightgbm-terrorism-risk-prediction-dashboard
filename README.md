# Lightgbm-terrorism-risk-prediction-dashboard
Built a LightGBM-based machine learning model to predict terrorism attack success using the Global Terrorism Database (GTD). Developed an interactive Streamlit dashboard for real-time predictions and visualization of global terrorism patterns.
